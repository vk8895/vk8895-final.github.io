Data Analysis Process



Step 1: Gather Data

Used Quandl API to pull stock quotes. Used Dow Jones Index to track large companies with sufficient stock data, across several major industries


Step 2: Build Technical Indicators

Add features from previous page into dataframe for stocks. Some of them are displayed here as an example.


Step 3: Run ML Models

Start with single and multivariable linear regression to identify significant features that impact the daily closing stock prices


Step 4: Try Predictions

If possible with existing features, try to use models to predict future stock prices. This would be the eventual goal of using ML for stock trading.
